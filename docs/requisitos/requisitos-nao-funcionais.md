# Requisitos Não Funcionais

- RNF01: o sistema deve possuir autenticação segura
- RNF02: o sistema deve utilizar HTTPS
- RNF03: o sistema deve possuir isolamento de dados entre escritórios
- RNF04: o sistema deve utilizar banco de dados robusto
- RNF05: o sistema deve possuir testes automatizados
- RNF06: o sistema deve possuir pipeline de integração contínua
- RNF07: o sistema deve seguir princípios de Clean Code e SOLID
- RNF08: o sistema deve ser preparado para execução em ambiente de nuvem
